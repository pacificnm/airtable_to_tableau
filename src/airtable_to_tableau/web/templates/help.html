🆘 Help & Documentation

Welcome to the help section. Here you’ll find usage tips, CLI examples, and troubleshooting steps.

🛠 CLI Usage

airtable-export export --config configs/yourfile.json --profile default

📁 File Types

  • .json — Config files for exporting
  • .hyper — Tableau Hyper output files

❓ Troubleshooting

If you encounter errors, check:

  • That your AIRTABLE_API_KEY is set
  • The config file path is correct
  • The base ID and table name match your Airtable workspace
